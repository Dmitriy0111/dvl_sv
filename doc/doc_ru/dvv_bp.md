# dvv_bp (base port)

Данный файл содержит описание класса базового порта. Данный класс не может иметь экземпляры, так как используется модификатор virtual, но от него производится наследование. Классы dvv_ap (analysis port) и dvv_aep (analysis export) наследуются от dvv_bp.

## Поля и функции/задачи класса  

Класс содержит следующие поля:
| Имя       | Тип                 | Описание                    |
| --------- | ------------------- | --------------------------- |
| p_name    | string              | Имя порта                   |
| bp_list   | dvv_bp #(item_type) | Список портов приёмников    |

Класс содержит следующие функции/задачи:
| Имя       | Описание                              |
| --------- | ------------------------------------- |
| write     | Функция для приёма/передачи           |
| connect   | Задача для соединения портов          |
