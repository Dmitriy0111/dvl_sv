# dvv_res (resource class)

Данный файл содержит описание класса ресурсов. 

## Поля и функции/задачи класса 

Класс содержит следующие поля:
| Имя       | Тип       | Описание                          |
| --------- | --------- | --------------------------------- |
| res_val   | res_t     | Значение ресурса с заданным типом |
| res_name  | string    | Имя ресурса                       |

Класс содержит следующие функции/задачи:
| Имя           | Описание                                                          |
| ------------- | ----------------------------------------------------------------- |
| new           | Конструктор класса                                                |
