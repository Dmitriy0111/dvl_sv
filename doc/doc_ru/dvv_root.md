# dvv_root (root class)

Данный файл содержит описание главного верификационного класса. 

## Поля и функции/задачи класса 

Класс содержит следующие поля:
| Имя          | Тип        | Описание                                              |
| ------------ | ---------- | ----------------------------------------------------- |
| domain       | dvv_domain | Верификационный домен для запуска этапов тестирования |
| test_name    | string     | Имя теста                                             |

Класс содержит следующие функции/задачи:
| Имя       | Описание                                 |
| --------- | ---------------------------------------- |
| new       | Конструктор класса                       |
| run_test  | Задача для запуска процесса тестирования |

Кроме этого данный класс также содержит поля и функции/задачи dvv_bc класса.
